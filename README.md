# Financial Organizer

This is a simple financial tracker user interface (UI) built using Java Swing. It allows users to input their financial transactions and view summaries of their expenses. This README file will guide you through setting up and running the application.

### Requirements:

- Java Development Kit (JDK) installed on your system.
- Basic understanding of Java Swing.

### Setup:

1. Clone the Repository:

    ``` git clone https://github.com/MeGaTroNOO7/Fin-Org.git ```

2. Compile and run the Java Files
